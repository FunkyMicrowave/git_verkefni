Git spurningar og svör
1. Hvað gera eftirfarandi Linux skipanir?
	cd     breytir hvar þú ert
	cd ~   uppfærir þar sem að þú ert núna
	ls     Skoðar hvað er á staðnum sem að þú ert núna
	pwd    Segir hvar staðsetningin er núna
	mkdir  Býr til möppu

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone      "klónar" skrána
	git log        skoðar hvað er búið að gera, skoðar history
	git status     skoðar hvað er búið að gera
	git diff       skoðar tvo file-a og prentar út hvað er mismunandi
	git checkout   uppfærir file-ana þannig að þeir verði eins og á síðunni
    
3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff                    Skoðar folderinn þar sem að þú ert núna
	b) git diff --staged           Skoðar folerinn
	c) git diff commit1 commit2    Skoðar  file-ana sem að er skrifaðir inn

4. 	Hvað er útgáfustýring (e.version control)? 
    Fer yfir hvaða file er nýjastur með öllu uppfærðu

5.	Hverjir eru helstu kostir við að nota GIT?
    Auðvelt að vinna með einhverjum því git breytir því sem að er ekki eins í file-num sem að var að koma inn á síðuna
    
6.	Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu?
    eins oft og mögulegt er, því að file-inn uppfærist í hvert skipti sem að er commitað þannig að ef að hinn aðillinn er að vinna á sama tíma þá eruð þið alltaf að uppfæra það sem að hinn aðillinn gerir
    

7.	Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT?
    a) Á hvaða local svæði þú ert að vinna
    b) Á hvaða svæði folderarnir fara eftir að það er búið að pusha þeim
    c) Á hvaða server ert þú ert að vinna á
    
8.	Hvenær er sniðugt að nota greinar (branches)? 
    Þegar þú ert að vinna verkefnið og veist að hinn aðillinn er að vinna á sama tíma
